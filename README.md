## 📖 О проекте

Это одностраничное портфолио, созданное для демонстрации моих навыков в области дизайна. Сайт выполнен в минималистичном стиле с акцентом на визуальную составляющую и удобство навигации.

**Особенности:**
- Адаптивная вёрстка под все типы устройств
- Горизонтальный слайдер с карточками навыков (прокрутка колёсиком мыши)
- Плавное размытие вступительного блока при скролле
- Летающие «плашки» с информацией о возрасте и опыте
- Тёмная контактная секция с фоновыми иконками социальных сетей
